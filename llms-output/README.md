## Usage

This is a React/Preact UI component library built with TypeScript and CSS Modules.

### Installation

```bash
npm install @object-ui/components
```

### Basic Usage

```tsx
import { Button } from '@object-ui/components';

function App() {
  return <Button onClick={() => {}}>Click me</Button>;
}
```
