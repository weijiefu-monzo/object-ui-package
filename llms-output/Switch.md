# Switch Component

## Props

| Property | Type | Required | Description |
|----------|------|----------|-------------|
| checked | boolean | Yes | - |
| onChange | (checked: boolean) => void | Yes | - |

## Examples

### Default

```tsx
<Switch
  checked="false"
/>
```

### Checked

```tsx
<Switch
  checked="true"
/>
```

### Interactive

```tsx
<Switch
  checked="false"
/>
```
